About the Game

Om Nom Run is an exciting endless runner game where you control the adorable Om Nom character. Jump over obstacles, slide under barriers, and collect delicious candies as you run through various challenging levels. With its simple controls and addictive gameplay, Om Nom Run offers endless entertainment for players of all ages.

Game Features

  • Simple one-touch controls
  • Multiple challenging levels
  • Collect candies and power-ups
  • Beautiful graphics and animations
  • Endless replay value

Play Now

How to Play

  1. Click the game window to start
  2. Use spacebar or tap to jump
  3. Slide down to avoid obstacles
  4. Collect candies for extra points
  5. Try to achieve the highest score!
